BasicGUI

PURPOSE

This class produces a basic graphical interface. The minimalist design

SYNOPSIS

This is a script file.

DESCRIPTION 

 This class produces a basic graphical interface.  The minimalist design
 of this interface allows it be be useful across different
 hardware/sofware systems.  The design combines elements that resizable
 elements (things that scale with figure size) with fixed elements.

 To generate a new graphical interface:
    object=BasicGUI(...)
 where input arguments are passed to the Figure class.  Creating this
 object generates new MATLAB figure containing three things: a panel for
 axes object, a panel for push buttons, and a custom toolbar (generated by
 the Figure class).  No menus are associated with the figure, though they
 can be added as necessary.  

 A panel for push buttons is located at the bottom of the figure.  The
 axes panel is located directly above the button panel.  Most of the
 figure is allocated to the axes panel; the button panel is only as large
 as needed to contain the defined buttons.

 See also SMASH.MUI, Figure
